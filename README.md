An E-commerce website made with Laravel framework (V-6).

## Usage

1. Clone this repository
2. `composer install`
3. `cp .env.example .env`
4. `Config DB info in .env`
5. `php artisan key:generate`
6. `php artisan migrate`
7. `php artisan db:seed`
8. `php artisan serve`
